# Security Policy

## Supported Versions

Please refer to the [Compatibility Matrix](COMPATIBILITY.md) for the current status of version

## Reporting a Vulnerability

If you happen to find a vulnerability, please open an Issue detailing what and how.  You can also send me a DM at any time if you feel posting an Issue would endanger other users.  (I REALLY hope that this code can't be THAT vulnerable, but well, just in case!)

Thank you for showing responsibility to the greater community!
